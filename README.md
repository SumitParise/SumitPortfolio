# Sumit Parise | Software Engineer Portfolio

A premium, interactive personal portfolio website for Sumit Parise, Software Engineer. Built using React, TypeScript, Tailwind CSS, GSAP for smooth timeline animations, and React Three Fiber (Three.js) for the background 3D particle and geometry scenes.

## Features
- **Dynamic 3D Canvas**: Interactive wireframe geometry and particle system responsive to mouse movement.
- **GSAP Animations**: ScrollTrigger-based staggered reveals and custom timeline drawings.
- **Custom Cursor**: A lag-following cursor ring that changes state on hover.
- **Breathing Aurora**: CSS-animated radial gradients for a premium visual aesthetic.
- **Mobile Responsive**: Fully optimized for mobile, tablet, and desktop viewports.

## Tech Stack
- **Frontend**: React (TypeScript), Vite
- **Styling**: Tailwind CSS, Vanilla CSS Variables
- **Animations**: GSAP & `@gsap/react`
- **3D Graphics**: Three.js, `@react-three/fiber`, `@react-three/drei`
